Git

  • add - Добавляет файл или файлы в индекс, готовит их для коммита.
  • rm - используется для удаления файлов из из индекса и рабочей среды
  • commit - Формирует новый коммит с комментарием, описывающим суть внесённых правок.
  • branch - Управляет ветками в репозитории: создаёт новые ветки, а также просматривает, переименовывает и удаляет существующие.
  • push - Отправляет изменения из локального репозитория в удаленный.
  • mv - расфшифровывается как"Move", что переводится как "переместить", "передвинуть". С помощью нее можно переместить и переименовать файл
